## Description

Exploring functionality of [Nest](https://github.com/nestjs/nest) framework.
Based on [API with NestJS](https://wanago.io/courses/api-with-nestjs/) series.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
